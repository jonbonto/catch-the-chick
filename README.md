# 🐤 Catch the Chick!

A fun and simple browser-based game where you catch falling baby chicks with a basket!

## 🎮 Game Concept

Baby chicks are falling from the sky! Use your basket to catch as many as you can before they hit the ground. Don't let too many fall, or it's game over!

## 🕹️ How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin
3. Use the **Arrow Keys** (← →) or **A/D keys** to move the basket left and right
4. Catch the falling chicks 🐤 with your basket 🧺
5. Don't let more than 5 chicks fall!

## 🎯 Game Features

- **Simple Controls**: Easy to learn, fun to play
- **Progressive Difficulty**: The game gets harder as you level up
- **Score System**: Earn 10 points for each chick you catch
- **Level System**: Level up every 50 points
- **Cute Visuals**: Adorable chick and basket emojis

## 📊 Scoring

- **Catch a chick**: +10 points
- **Miss a chick**: +1 missed (max 5)
- **Level up**: Every 50 points

## 🎨 Features

- Responsive design
- Smooth animations
- Cloud effects in the background
- Pause/Resume functionality (Space bar)
- Game Over screen with final score

## 🚀 Getting Started

Simply open the `index.html` file in any modern web browser. No installation or build process required!

```bash
# Just open the file directly or use a simple HTTP server
python3 -m http.server 8000
# Then navigate to http://localhost:8000
```
